About This 4K Recreation

This page presents two independent 3D reconstructions of the St. Xavier's Junior Senior School crest, both rendered at 4K resolution (3840×3840) from the same reference image. Click any model to open it in fullscreen mode — drag to rotate, scroll to zoom, right-click to pan.

Method 1: Blender Headless (High-Detail)

The Blender model was built with a Python script in headless Blender 3.6 LTS. The reference image was analyzed pixel-by-pixel (k-means color clustering, connected-component segmentation) and a VLM identified every element's exact position, size, and color. Each element is a separate mesh: gold shield frame (bmesh annulus), blue field (filled polygon), 16-ray sunburst, India map + Sri Lanka, gold scales of justice with chains and pans, torch with brown handle + gold cup + 3-layer orange-yellow flame, V-shaped open book with text lines, curved red banners with extruded serif text. Rendered with Cycles CPU at 96 samples, 2K native then upscaled to 4K with LANCZOS.

Method 2: Blockbench Voxel (Browser Workbench)

The Blockbench model uses 1,695 colored cubes on a 128×128 grid. Each crest element is built from multiple cubes for finer detail — the shield frame, blue field, 16 sunburst rays, India landmass, scales, torch with flame layers, open book with text lines, curved banners, and text marks. Editable in the Blockbench web editor. Rendered as a 4K top-down projection.

Element Inventory (Both Models)

  • Blue shield field — royal blue interior (#085999)
  • 16-ray golden sunburst — alternating long/short rays

  • Torch — brown handle, skin hand, gold cup, orange→yellow flame
