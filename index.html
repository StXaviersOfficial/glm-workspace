How This Works (Image-to-3D)

This is a true image-to-3D bas-relief reconstruction — the same approach used by Meshy.ai, but done locally with open-source tools. The original crest image is used both as the surface texture AND as the source for 3D displacement depth. The result is a 3D-printable model that looks exactly like the reference when viewed from the front, with real geometric depth.

Process

  1. Height map generation: The reference image is analyzed using luminance + saturation + edge detection + background-difference. Colored crest elements get high values (raised), white background gets low values (flat). Smoothed with Gaussian blur.
  2. Mesh subdivision: A flat plane is subdivided into a high-poly grid (256×256 for Blender = 65,536 vertices, 200×200 for trimesh = 40,000 vertices).
  3. Vertex displacement: Each vertex's Z coordinate is set by sampling the height map at its UV position. This creates real 3D relief geometry — the torch, scales, book, and banners all have actual depth.
  4. UV mapping + texture: The original crest image is UV-mapped onto the displaced surface. The front face shows the exact image, warped onto the 3D relief.
  5. Extrusion for 3D printing: The entire mesh is extruded backward (Solidify modifier in Blender, manual extrusion in trimesh) to give it physical thickness — ready for 3D printing as a coin/medallion/bas-relief plaque.
  6. GLB export: The final mesh with embedded texture is exported as GLB, viewable in any 3D viewer including model-viewer on this page.

Why this is better than primitive-based 3D

Previous attempts tried to recreate the crest using 3D primitives (cubes, spheres, cylinders) — this always looks "blocky" and never matches the exact image. The bas-relief approach uses the actual image pixels as the surface, so every detail (text, fine lines, color gradients) is preserved exactly. The 3D depth comes from the height map, giving real geometric relief that catches light and shadow naturally.

3D Printing

Both GLB files are 3D-printable. The mesh is manifold (watertight) with proper thickness. Print settings: 0.2mm layer height, PLA/PETG, ~80mm × 80mm × 8mm. The front face shows the full-color crest (if using a color 3D printer), the back is flat. For single-color printing, the relief depth creates visible shadows that reveal all details.
